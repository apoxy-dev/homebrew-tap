# Homebrew Tap for Apoxy

This is the official Homebrew tap for the [Apoxy CLI](https://apoxy.dev).

## Installation

To install Apoxy via Homebrew, run:

```bash
brew install apoxy-dev/tap/apoxy
```

## Usage

After installation, you can use the `apoxy` command:

```bash
apoxy --help
```

For comprehensive documentation on how to use Apoxy, including configuration, examples, and advanced features, visit the [Apoxy documentation](https://apoxy.dev/docs).

## What is Apoxy?

Apoxy is a command-line tool for managing and deploying networking services. Visit [apoxy.dev](https://apoxy.dev) to learn more.

## Updates

To update Apoxy to the latest version:

```bash
brew upgrade apoxy
```
